============
Installation
============

The easiest way to install MyAutoML is to install it from PyPI.

Installing from PyPI
~~~~~~~~~~~~~~~~~~~~

MyAutoML can be installed via pip from
`PyPI <https://pypi.org/project/myautoml>`__.

.. code-block:: bash

    pip install myautoml
